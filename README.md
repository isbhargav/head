# Byte Garden

A digital garden built with Hugo and the Zen theme.

* View the site: <https://byte-garden.xdeb.org>
* Download theme at: <https://github.com/frjo/hugo-theme-zen>

Feel free to use this repo as a template if you want it as a quickstart for your site. Use the green "Use this template" button above.

## Features

* GitHub pages site.
* Auto deployed on push via GitHub actions, see [workflow file](https://github.com/frjo/byte-garden/blob/main/.github/workflows/pages.yaml).
* Zen theme loaded as a Go module (update with `hugo mod get -u`).
* Multilingual
* Search
* Podcast
