A collection of tweet-like notes with sorting, searching and filtering capabilities using [List.js](https://listjs.com/) that is included with the Zen theme.

There are some js and html code needed for each implementation. See [listsort.js](https://github.com/frjo/byte-garden/blob/main/assets/js/listsort.js) and [notes/list.html](https://github.com/frjo/byte-garden/blob/main/layouts/notes/list.html)
